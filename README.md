No overlap smart ressurection. Handles mouse clicks and keybinds.

Left click to ress.  
Right click to skip to next corpse.  
`/szcz help` for commands.  
  
Default setting for res is furthest in range. If you want closest, edit:  
`SPELL_DISTANCE_MODE = "furthest"`
